# spade-lsp

Registers the [Spade HDL](https://spade-lang.org) language server with Claude
Code's `LSP` tool for `.spade` files.

The plugin maps `.spade` to the `spade-language-server` command, which must be
on your `PATH`. See the [repository README](../../README.md) for prerequisites,
installation, and the `swim lsp` alternative.
